## Intellectual Property & Trust

We take client IP seriously:

- **Your schematics are yours.** TFC never claims ownership of client `.litematic` files or designs.
- **Secure handling.** Files are securely stored with limited internal access during active projects.
- **Deleted on completion.** All client schematic files are removed from our systems when a project is finished.
- **Zero tolerance.** Unauthorized sale, redistribution, or reuse of client files results in immediate staff termination.
